Spring MVC Showcase
-------------------
小实例演示sprigmvc用法

包括:

* @Controller用法
* 请求地址映射
* 返回数据处理
* 生成响应
* 消息转换
* 视图渲染
* 类型转换
* 数据校验
* 表达数据收集
* 文件上传
* 异常处理

-------------------	
maven运行项目:

    $ cd spring-mvc-showcase
    $ mvn jetty:run 

访问地址: http://localhost:8080/spring-mvc-showcase/
